IPv6RegTweakGUI

Application to make it easy to configure several advance Microsoft Windows IPv6 settings.

View on GitHub Download .zip Download .tar.gz

What is it?

IPv6RegTweakGUI(working title) is a portable application to allow to enable or disable several IPv6 transition technologies in Microsoft Windows. Normally to disable these technologies the Windows registry has to be used but this application allow the user to disable or enable several IPv6 transition technologies by just using several checkboxes.
